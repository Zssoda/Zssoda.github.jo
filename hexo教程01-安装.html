Hexo安装

Hexo安装基本流程

Hexo的安装十分简单，但在安装前，必须要求我们安装Node.js和Git，因此其安装仅需要以下三步

  1. 安装Git 下载并安装
  2. 安装Node.js 下载并安装
  3. 安装Hexo 打开Git bash,输入

因为是记录一路的学习过程，下面几小节比较琐碎，阅读完第一节可以顺利安装的同学，即可跳过，如遇问题，可在下面几小节查阅。

安装Git

Git介绍

Git是目前世界上最先进的分布式版本控制系统，方便我们对多个版本进行管理。具体介绍可见廖雪峰Git教程

博客搭建过程建议都是用git来进行版本控制。目前hexo使用初期，主要感受就是由于Hexo的很多操作都涉及到命令行，而Git bash是Linux风格shell,因为习惯Linux命令所以都使用Git Bash来进行操作hexo的操作。

windows下Git安装

下载地址：

  • Git
  • Git其他下载地址


点击安装即可，检查是否安装成功，任意位置右键，查看是否有“Git Bash Here”，选择打开。

其他平台Git安装

  • Linux (Ubuntu, Debian)：sudo apt-get install git-core

  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

  • Mac：使用 Homebrew, MacPorts ：brew install git;或下载安装程序

安装Node

Node介绍

Node是一个基于GoogleV8引擎的运行在服务端的框架，具体介绍可以参见博文《为什么要用Node.js》和《Node.js到底是什么》)

在hexo的使用.Node.js最直观的作用就是与其绑定的npm工具,该工具可以方便下载和安装第三方包和命令行程序以供使用，在后期使用Next进行hexo优化时，常常会用到，感兴趣可以戳这个链接提前了解一下用法。
